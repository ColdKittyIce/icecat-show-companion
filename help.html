IceCat Show Companion

Help Guide v3.0

Everything you need to run a smooth show on Prankcast — soundboards, session logging, mic control, and more.

🚀 Quick Start

New to the companion? Get up and running in five steps.

1

Set up your show details

Open Settings → Show. Enter your show name, Prankcast URL, and episode number. These appear in session log exports.

2

Configure your audio devices

Go to Settings → Audio. Select your output device (soundboard playback), input device (recorder capture), and mic input device (for the mute button). If you use Voicemeeter, your virtual devices will appear here.

3

Load your soundboard

Right-click any soundboard button and choose Assign File to load a sound. Drag & drop audio files directly onto buttons. Right-click a bank tab to change its colour.

4

Go live

Click GO LIVE (or press Ctrl+Shift+L) when your stream starts. The ON AIR badge lights red, the live timer starts, and session log timestamps include your elapsed time.

5

End your show

Click END LIVE to stop the timer. Use Session Log → 💾 Export to save a full show log. Your config auto-saves — everything will be exactly where you left it next time.

Interface Overview

The companion window is divided into four main zones:

Header & Broadcast Controls

The header runs across the full width and gives you everything you need to manage the broadcast itself.

GO LIVE / END LIVE

Toggles the broadcast state. When live: the ON AIR badge turns red, the live timer counts up, and all session log entries include [elapsed] time. Right-click to rename or recolour the button.

CALL Timer

Click 📞 Start to begin timing a caller. The timer counts up in MM:SS. Click ⏹ End Call to stop — the start and end events are automatically stamped to the Session Log including the total call duration.

COUNTDOWN

Type a time in MM:SS and press ▶, or use the 5m / 10m quick buttons. The display turns red in the final 30 seconds. Useful for segment timing or music bed fade-outs.

VOL / MUTE / PANIC

  • VOL — master volume for all soundboard and queue playback.
  • MUTE — silences all audio instantly. Click again to restore. Right-click to recolour.
  • PANIC — stops every sound playing immediately. Emergency use. Also bound to Ctrl+Shift+P.

Soundboard

The soundboard is the heart of the companion. Buttons are arranged in banks, each with configurable rows and columns.

Assigning sounds

  • Right-click any button → Assign File to browse for an audio file.
  • Drag & drop an audio file from Windows Explorer directly onto any button.
  • Supported formats: .mp3 .wav .ogg .flac .m4a

Playing sounds

  • Left-click — play the sound. Click again to stop (toggle mode).
  • Each button gets its own audio channel — sounds play simultaneously.
  • Buttons turn blue while playing, return to normal when finished.

Bank management

  • Click a bank tab at the top to switch banks instantly (cached — no loading delay).
  • Click + Bank in the tab bar to add a new bank on the fly — name it and it's ready immediately.
  • Right-click a bank tab → change colour, export/import the whole bank as JSON.
  • Use Settings → Soundboard to rename banks, change rows/columns, or delete banks.

Pinned row

The pinned row sits above the bank tabs and is always visible regardless of which bank you're on — ideal for sounds you need in every segment (e.g. your intro sting, applause, hold music).

Edit mode

Click 🔓 Edit in the tab bar to unlock the soundboard. In edit mode you can drag files, rename buttons, and right-click for full options. Click 🔒 Locked to prevent accidental changes during a live show.

💡 Tip Right-click any assigned button to access FX settings — add reverb, pitch shift, or a loopback echo per-button. FX apply on the next play.

Music Queue

The queue sits to the right of the soundboard and handles your background music playlist.

Adding tracks

  • Click Browse & Add to open a file picker (multi-select supported).
  • Drag & drop audio files directly into the queue list.

Reordering

Grab the ⠿ drag handle on the left of any queue row and drag it to a new position. The dragged item highlights amber and a line shows the insertion point.

Auto mode

Toggle Auto to enable continuous playback — when one track ends, the next starts automatically. Perfect for unattended music beds.

VOL/FADE slider

The vertical slider beside the queue controls the performance fade level — separate from master volume so you can duck music without affecting soundboard effects. The Mic panel's 🎚 Duck button uses this slider.

Loading files

Drag any audio file from Windows Explorer onto the recorder section to load it for playback. Useful for playing back a previous recording or a pre-recorded segment.

Recordings list

Click the dropdown below the transport to switch between saved recordings. Files are stored in your configured recordings folder (default: %USERPROFILE%\IceCatCompanion_v3\recordings\).

📝 Note Set your input device in Settings → Audio → Input Device before recording. For a full show recording, use Voicemeeter Output or Stereo Mix to capture everything.

Mic Panel

The mic panel sits between the PANIC button and the tape recorder in the header.

🎙 LIVE / 🔴 MUTED

Mutes or unmutes the Windows input device you selected in Settings → Audio → Mic Input Device. Sets both the Windows mute flag and the volume to zero for a hard mute. On app close, the mic is always restored to unmuted.

🎚 Duck (toggle)

Press once to smoothly fade the VOL/FADE slider down to your configured duck level. Press again to fade it back up. The fade speed matches your Settings → Audio → Fade Duration. Button turns amber when ducked.

GAIN slider

Controls the Windows input device volume (0–100%). This is the mic gain at the OS level.

📝 Setup required Go to Settings → Audio → Mic Input Device and select your microphone for the mute button to work. If the label shows 🎙 NO DEVICE, the device hasn't been configured.

Right Panel

The right panel contains quick access tools stacked vertically.

Quick Folders + Sites

Six programmable folder buttons in two rows. Right-click any button to assign a folder path or customise the label and colour. Click the button to open that folder in Windows Explorer.

Below the folder buttons is a website dropdown. Select a site and click Go to open it in your browser. Use + to add a site, 🗑 to remove the selected one. Full management is in Settings → Websites.

Notes

Located in the bottom-left strip. Two tabs by default: Show Notes and Premises & Ideas (configurable in Settings → Show).

  • 🗑 Clear — clears the current tab (asks for confirmation).
  • 💾 Save — opens a Save As dialog to export the current tab to a .txt file.
  • Notes auto-save to config every 5 minutes and on app close — you won't lose your notes.

Session Log

The session log automatically records everything that happens during your show.

What gets logged automatically

  • Every sound played (bank name, button label, duration)
  • Queue tracks played
  • GO LIVE / END LIVE events
  • Call started / Call ended (with duration)
  • Gold moment markers

Timestamp row

At the top of the log area: type a note in the text field and press Enter or click 📌 Timestamp to add a custom stamped entry. Leave the field empty to add a plain timestamp with no note.

Timestamp format

Entries show: Mar 22 14:32:01. When live, they also include elapsed show time: Mar 22 14:32:01 [01:23:45].

Bits Board

A scratchpad for your show bit ideas — title, notes, and a done state.

  • ◀ / ▶ — navigate between bits.
  • + Add Bit — creates a new blank bit at the end.
  • 💾 Save Bit — saves the current bit's title and content to config.
  • ✓ Mark Done — marks the bit as completed (text dims). Auto-saves.
  • 🗑 Delete — removes the current bit (must have at least one).
  • ↺ Reset — clears all Done marks across all bits.
  • 🎲 — jump to a random bit (great for when you're stuck for ideas).

Mini Mode

A compact always-on-top strip for when you need the companion visible but out of the way. Activate with Ctrl+Shift+Z or via the View menu.

Drag the mini window by the title bar. It stays on top of all other windows. Click ⛶ Expand or press Ctrl+Shift+Z again to return to the full window.

Settings — Show

  • Show Name — appears in session log exports and the title bar.
  • Episode Number — auto-increments each time you click GO LIVE. Used in title templates.
  • Title Template — e.g. The Chill With IceCat #{n} — {date}. Variables: {n} = episode number, {date} = today's date.
  • Prankcast URL — used for the Discord webhook go-live notification.
  • Note Tabs — comma-separated tab names for the Notes section.

Settings — Audio & Mic

  • Output Device — where soundboard audio plays. Select your Voicemeeter virtual input or speakers.
  • Input Device (recorder) — what the tape recorder captures. Use Voicemeeter Output or Stereo Mix for full show recording.
  • Recordings Folder — where saved recordings are stored.
  • Recording Format — WAV (lossless) or MP3 (compressed).
  • Fade Duration — how long the VOL/FADE fade takes in seconds (used by Duck button too).
  • Mic Input Device — the Windows input device the Mute button controls. Pick your physical mic or Voicemeeter strip.
  • Duck Level — how far the VOL/FADE slider drops when you press Duck (0% = silent, 100% = unchanged).

Settings — Soundboard

  • Pinned Row Count — how many buttons in the always-visible pinned row (2–12).
  • Music Bank — designate one bank as the music bank for more detailed session log entries.
  • Banks — rename banks, set rows/columns, delete banks. Changes take effect on Save & Close.

Settings — Hotkeys

All hotkeys use the format modifier+key, e.g. ctrl+shift+l. Leave blank to disable a hotkey.

Settings — Websites

Add, edit, or remove websites that appear in the Quick Folders + Sites dropdown. Enter a short Label and the full URL (including https://). Changes take effect immediately on Save & Close.

Settings — Visual

  • Color Theme — choose from Default Blue, Midnight Purple, Ice Blue, or Matrix Green. Requires app restart.
  • Font Size — base font size for the interface.
  • Window Opacity — make the window semi-transparent (useful for streaming overlays).

💡 Tips & Tricks

⚡

Instant bank switching

Sound durations are cached after first load. Bank tabs switch with zero delay after the first visit.

🎚️

Duck for callers

Hit 🎚 Duck in the mic panel before putting a caller to air — music fades smoothly. Hit it again when they hang up.

📌

Stamp everything

Use the timestamp field in the session log to mark your best bits. When you export, you have a timestamped show map.

🎲

Bits board randomiser

Click 🎲 when you're running dry on ideas mid-show. It jumps to a random unread bit.

🔒

Lock before going live

Click 🔒 Lock in the soundboard tab bar before you start. Prevents accidentally moving or overwriting a button mid-show.

📻

Mini mode for dual screen

Run the full companion on your production screen. Pop mini mode onto your second monitor for on-air use — it's always on top.

💾

Config backups

The app auto-backs up your config on every launch (keeps last 5). For a manual backup before big changes, use Settings → About → 🔒 Backup Config.

🎵

Queue auto mode

Load your whole music library into the queue and turn on Auto. It plays through continuously without any intervention.

🔧 Troubleshooting & FAQ

The app won't start — I see a Python error

Run pip install -r requirements.txt in the app folder to ensure all dependencies are installed. Key ones: customtkinter, pygame, sounddevice, pycaw, tkinterdnd2.

Sound buttons aren't playing audio

Check Settings → Audio → Output Device. If it says "Default (System)" and you're using Voicemeeter, select your Voicemeeter virtual input instead. Also check the VOL slider isn't at zero and MUTE isn't active.

The Mic Mute button shows "NO DEVICE"

Go to Settings → Audio → Mic Input Device and select your microphone from the dropdown. The list shows proper Windows device names — look for your physical mic or the Voicemeeter strip you use for your mic. Save & Close to apply.

The mic is muted after closing the app

This is fixed in v3.0 — the app always restores the mic to unmuted on close. If you're on an older build, update to the latest zip.

Drag and drop onto soundboard buttons isn't working

Make sure tkinterdnd2 is installed (pip install tkinterdnd2). Also try dragging from Windows Explorer rather than another app. The button must be in Edit mode (🔓 Edit in the tab bar).

The device dropdown in Audio settings shows weird names or duplicates

The app uses the Windows MMDevice API for device names. If you see duplicates, one is likely a legacy driver entry — pick the one that matches what you see in Windows Sound settings. Voicemeeter devices are sorted to the top.

Session log timestamps don't show elapsed time

Elapsed time only appears in brackets when you're live (GO LIVE pressed). Off-air entries show wall-clock time only.

My notes disappeared

Notes auto-save to config.json every 5 minutes and on close. If you renamed a notes tab in Settings, the content for the old tab name is still in config under the old key. Check %USERPROFILE%\IceCatCompanion_v3\config.json in a text editor under "notes_content".

How do I reset everything to defaults?

Close the app, then delete %USERPROFILE%\IceCatCompanion_v3\config.json. The app will regenerate fresh defaults on next launch. Your recordings are unaffected.

Where are my files stored?
  • Config: %USERPROFILE%\IceCatCompanion_v3\config.json
  • Config backups: %USERPROFILE%\IceCatCompanion_v3\config_backups\
  • Recordings: %USERPROFILE%\IceCatCompanion_v3\recordings\
  • Error log: %USERPROFILE%\IceCatCompanion_v3\logs\icecat.log
